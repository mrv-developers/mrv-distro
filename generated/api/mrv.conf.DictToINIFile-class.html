Wraps a dictionary into an objects returning an INI file when read

This class can be used to make configuration information as supplied by os.environ natively available to the configuration system

Notes:
  • writing back values to the object will not alter the original dict
  • the current implementation caches the dict's INI representation, data is not generated on demand
  • implementation speed has been preferred over runtime speed
