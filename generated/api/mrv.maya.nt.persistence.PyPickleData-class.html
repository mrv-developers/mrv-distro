Class PyPickleData

source code

OpenMayaMPx.MPxData --+
                      |
                     PyPickleData

Allows to access a pickled data object natively within a maya file. In ascii mode, the pickle will be encoded into string data, in binary mode the cPickle will be taken in its original value.

To get the respective dict-references back, we use a tracking dict as proposed by the API Docs

Notes:
  • This datatype is copies the data by reference which is why maya always calls the copy constructor, even if you retrieve a const data reference, where this would not be required actually. This is fine for most uses
  • as the datatype is reference based, undo is currently not supported (or does not work as it is expected to do
